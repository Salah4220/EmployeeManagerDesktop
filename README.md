# EmployeeManagerDesktop

Application WPF connectée à une API REST ASP.NET Core, pour gérer les employés.

## Projets

- **EmployeeManager.Api** – API REST sécurisée (JWT) + PostgreSQL
- **EmployeeManager.Desktop** – Application desktop WPF (MVVM)
- **EmployeeManager.Shared** – DTOs et types partagés

## Démarrage rapide

```bash
docker-compose up --build
